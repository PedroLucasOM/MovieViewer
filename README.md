> Angular interface for searching and viewing movie information through The Movie DB API.

### 🏠 [Homepage](https://github.com/PedroLucasOM/MovieViewer)

## Prerequisites

- npm >=6.13.4
- node >=12.16.1
- @angular = 8.1.0

## Install

```sh
npm install
```

## Usage

```sh
ng serve
```
